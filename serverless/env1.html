Key Value
LAMBDA_TASK_ROOT Contains the path to your Lambda function code.
AWS_EXECUTION_ENV The environment variable is one of the lang supported by AWS
LAMBDA_RUNTIME_DIR Lambda runtime-related artifacts
AWS_REGION AWS region of function
AWS_LAMBDA_LOG_GROUP_NAME CloudWatch Logs group for log output
AWS_LAMBDA_LOG_STREAM_NAME CloudWatch Logs for STDOUT/STDERR
AWS_LAMBDA_FUNCTION_NAME The name of the Lambda function.
AWS_LAMBDA_FUNCTION_MEMORY_SIZE The size of the Lambda function in MB.
AWS_LAMBDA_FUNCTION_VERSION The version of the Lambda function.
TZ Defaults to UTC.
